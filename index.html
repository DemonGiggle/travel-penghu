澎湖 3 天旅遊規劃（7/6–7/8）

主題：海景放鬆、在地美食、花火節夜間行程。適合第一次到澎湖、節奏不趕的玩法。

3 天 2 夜 本島為主 含花火節

行前建議

  • 交通：建議早去午回（飛機優先，船班備案）。
  • 住宿：馬公市區，夜間移動與吃飯最方便。
  • 移動：機車最彈性；怕曬可考慮汽車。
  • 必備：防曬、防風外套、暈船藥、水壺。

預算抓法（每人）

  • 交通：NT$2,500–6,000（依城市/航班）
  • 住宿：NT$2,000–5,000（2 晚）
  • 餐食：NT$1,500–2,500
  • 租車/油資/門票：NT$1,000–2,000

總計約 NT$7,000–15,000 / 人

Day 1｜馬公市區 + 觀音亭花火節

  1. 上午：抵達澎湖 → 飯店放行李 → 市區午餐（海鮮粥/小管麵線）。
  2. 下午：篤行十村、中央老街、四眼井、天后宮散步拍照。
  3. 傍晚：觀音亭園區卡位（建議提早 60–90 分鐘）。
  4. 晚上：花火節主場觀賞 + 回市區吃宵夜（仙草、鹹酥雞、烤物）。

Day 2｜北環經典海景路線

  1. 上午：跨海大橋 → 通梁古榕 → 大菓葉玄武岩。
  2. 中午：西嶼一帶海鮮或在地麵店。
  3. 下午：池西岩瀑、二崁聚落、小池角雙曲橋看海。
  4. 傍晚：西嶼燈塔夕陽（風大請注意安全）。
  5. 晚上：回馬公自由活動（伴手禮採買、按摩放鬆）。

Day 3｜南海輕跳島 / 本島慢遊 + 回程

  1. 方案 A（跳島）：七美 + 望安一日（需早起，注意海況）。
  2. 方案 B（本島）：山水沙灘、隘門沙灘、林投公園悠閒收尾。
  3. 下午：回市區吃最後一餐，預留時間還車/登機。
花火節提醒： 本次行程日期為 7/6–7/8，實際施放日期/時間、無人機場次、是否因天候取消，請以「澎湖國際海上花火節官方公告」為準。
